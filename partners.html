Set up your Google Form (MVP)

Until we release the partner dashboard, the fastest way to accept submissions is via Google Forms feeding a Google Sheet that powers the public map.

  • Make a Google Form with fields: Partner name, Outlet address, Item name, Cuisine, Original price, Sale price, Halal (Yes/No), Veg (Yes/No), Quantity, Pickup start, Pickup end, Image URL.
  • Use Google Maps to get precise Latitude and Longitude for the outlet and include them as fields.
  • Link the form to a Sheet named Offers with matching columns.
  • Deploy the Apps Script in apps-script/Code.gs as a web app (Anyone access), then set your API URL in js/config.js.

Put your Google Form link in js/config.js so the “Become a partner” button opens it.
